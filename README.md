# Learning_Python_Day13

Today's topic is reading and writing files!

**Goal:** Master file operations to persist data beyond program execution

- **task1:** Create a text file and write 5 lines to it

- **task2:** Read the entire file content and print it

- **task3:** Read the file line by line using a loop

- **task4:** Append new content to the existing file

- **task5:** Use the `with` statement to automatically close the file

- **task6:** Read the file and count the total number of words

**mini-project:**
Simple Notes App:  
Create a small program that:

- Allows the user to add a new note (append to file)  
- Allows the user to view all saved notes (read file)  
- Counts and displays the total number of words in the file  
- Uses `with` for all file operations
